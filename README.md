# App Pages

This repository hosts public pages for mobile applications, including:

- Privacy Policy
- Support Pages
- Terms & Conditions
- App Information
- Landing Pages
- Other public resources

These pages are hosted using GitHub Pages and used for app publishing platforms such as the Apple App Store and Google Play Store.

## Structure

```text
/privacy
/support
/terms
```

## Hosting

Hosted with GitHub Pages.
